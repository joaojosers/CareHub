# Dev 3: Full-stack & Lógica de Negócio + DevOps/QA

## Foco Principal
Integração Front-to-Back, Regras de Negócio, Infraestrutura & QA

## Tecnologias
- React Hook Form (form handling)  
- Zod (validation)  
- Axios (HTTP client)  
- Docker (infraestrutura)  
- GitHub Actions (CI/CD)  
- Jest & Vitest (testing)  

## Responsabilidades
1. **Lógica de Negócio**  
   - Cálculo de horas, plantões, pagamentos  
2. **Integração Frontend-Backend**  
   - Conectar APIs com UI  
3. **Infraestrutura**  
   - Docker, GitHub Actions, CI/CD pipeline  
4. **Testing**  
   - Testes de integração, E2E, QA manual  
5. **Segurança Operacional**  
   - Security audits, OWASP checks  
6. **Deployment**  
   - Staging & production deployment  

## Success Metrics
- ✅ Todos os fluxos integrados funcionando  
- ✅ Testes E2E críticos passando  
- ✅ Security audit aprovado  
- ✅ Deployment sem incidentes
