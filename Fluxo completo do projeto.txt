🏗 FLUXO COMPLETO DO SISTEMA CAREHUB

🔐 1️. Administrador padrão
O sistema já inicia com um usuário:
·admin
·Exemplo: admin@email.com
·Acesso total ao sistema

✔ Funcionando

👩‍⚕️ 2️. Cuidador realiza cadastro
O cuidador se registra informando:
·Nome
·CPF
·Data de nascimento
·Telefone
·Endereço completo
·Email
·Senha

Status inicial:
status: "PENDENTE"
✔ Funcionando

✅ 3️. Admin aprova ou rejeita cuidador
Admin:
·Visualiza detalhes do cadastro
·Pode Aprovar
·Pode Rejeitar

Se aprovado:
status: "APROVADO"
✔ Funcionando

🔑 4️. Cuidador faz login
·Só pode logar se estiver "APROVADO"
·Se estiver "PENDENTE" ou "REJEITADO" → acesso bloqueado
✔ Funcionando

👨‍🦳 5️. Admin cadastra Paciente + Responsável
Admin cadastra:
·Paciente
·Familiar responsável 

O responsável já pode acessar o sistema imediatamente assim que cadastrado pelo admin.
✔ Funcionando

🔗 6️. Admin vincula cuidador ao paciente
Admin pode:
·Vincular 1 ou mais cuidadores ao paciente.
·Encerrar vínculo quando necessário
✔ Funcionando

⏱ 7️. Cuidador lança horas e relatório
Assim que estiver vinculado ao paciente, o cuidador pode:
·Lançar:
 Paciente
 Data do atendimento
 Horário início
 Horário fim
 Preencher relatório:
 Descrição das atividades realizadas
 Medicações administradas
 Pressão arterial
 Observações adicionais

Status inicial do lançamento:
status: "PENDENTE" (aguarda aprovação do admin)
✔ Funcionando

🧾 8️. Admin aprova horas + relatório
Admin visualiza lançamentos pendentes e pode:
·Aprovar → "APROVADO"
·Rejeitar → "REJEITADO"

Somente após aprovação:
✔ O responsável/familiar pode visualizar
✔ O relatório entra no histórico oficial
✔ Pode ser exportado

👨‍👩‍👧 9️. Responsável acompanha atendimento
O familiar:
·Visualiza somente relatórios "APROVADO"
·Vê:
 Horas trabalhadas
 Relatório detalhado
 Nome do cuidador
 Histórico por paciente
✔ Funcionando
